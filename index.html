Proyectos

🌩 Dashboard en tiempo real de caídas de rayos

Visualización interactiva de rayos a nivel mundial mediante datos en tiempo real extraídos de Blitzortung vía WebSocket. Los datos se almacenan en una base de datos SQL y se representan en Power BI, permitiendo explorar detalles como la ubicación, la hora, la polaridad y las estaciones que detectaron cada rayo.

Herramientas: Python (asyncio, websockets), Power BI, MySQL

🧾 Lectura automática de tickets de compra

Automatización que recibe fotos de tickets por WhatsApp, aplica OCR para extraer texto y utiliza ChatGPT para identificar productos, precios e IVA. Los datos se guardan en Google Sheets y se resumen en un dashboard.
